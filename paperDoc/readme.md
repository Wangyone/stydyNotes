# C++ 学习路线总结

##  第一阶段：计算机底层基础

在学 C++ 之前，先建立底层认知，否则很多概念会"知其然不知其所以然"。

**计算机组成原理**
- CPU 的工作原理（取指、译码、执行）
- 内存的层次结构：寄存器 → L1/L2/L3 缓存 → 内存 → 磁盘
- 地址总线、数据总线的概念

**操作系统基础**
- 进程与线程的概念
- 虚拟内存与物理内存、分页机制
- 栈（Stack）与堆（Heap）的区别与管理
- 系统调用的基本概念

**编译原理入门**
- 源码 → 预处理 → 编译 → 汇编 → 链接 的流程
- 目标文件、静态库（.a/.lib）与动态库（.so/.dll）
- 符号表与链接器的作用

---

##  第二阶段：C 语言基础（C++ 的根基）

> C++ 是 C 的超集，先掌握 C 能让你更理解底层行为。

- 基本数据类型、运算符、控制流
- 指针与数组（这是核心，必须彻底理解）
- 函数、递归、作用域
- 结构体 `struct`、联合体 `union`、枚举 `enum`
- 手动内存管理：`malloc` / `free`
- 文件 I/O 与标准库

---

##  第三阶段：C++ 核心语法

**面向对象（OOP）**
- 类与对象、构造/析构函数
- 访问控制：`public` / `protected` / `private`
- 继承（单继承、多继承）与虚函数
- 多态（运行时多态）与纯虚函数/抽象类
- 运算符重载

**内存管理（重点）**
- `new` / `delete` 与 `malloc` / `free` 的区别
- 深拷贝 vs 浅拷贝
- 拷贝构造函数、赋值运算符（The Rule of Three）
- 内存泄漏的排查思路（Valgrind 等工具）

**模板基础**
- 函数模板与类模板
- 模板特化

---

##  第四阶段：C++11 现代特性（核心）

C++11 是 C++ 的分水岭，必须重点掌握：

| 特性 | 说明 |
|---|---|
| `auto` / `decltype` | 类型推导，减少冗长类型声明 |
| 范围 `for` 循环 | `for (auto& x : container)` |
| 智能指针 | `unique_ptr` / `shared_ptr` / `weak_ptr`，告别裸指针 |
| 移动语义 | 右值引用 `&&`、`std::move`，性能关键 |
| 完美转发 | `std::forward`，配合模板使用 |
| Lambda 表达式 | 匿名函数，配合 STL 算法使用 |
| `nullptr` | 替代 `NULL`，类型安全 |
| 初始化列表 | `{}` 统一初始化语法 |
| `constexpr` | 编译期常量表达式 |
| 线程库 | `std::thread`、`std::mutex`、`std::async` |
| `std::function` / `std::bind` | 函数包装器 |

---

##  第五阶段：STL 标准模板库

- **容器**：`vector`、`list`、`deque`、`map`、`unordered_map`、`set` 等
- **迭代器**：理解迭代器模式，与算法解耦
- **算法**：`sort`、`find`、`transform`、`for_each` 等（配合 Lambda）
- **函数对象（仿函数）**：`std::less`、自定义比较器

---

##  第六阶段：进阶专题

按兴趣方向深入：

**模板元编程（TMP）**
- SFINAE、`std::enable_if`
- 可变参数模板 `template<typename... Args>`
- C++17 的 `if constexpr`

**并发编程**
- 线程安全、数据竞争、死锁
- 原子操作 `std::atomic`
- 内存模型与内存序

**设计模式与工程实践**
- RAII 原则（资源获取即初始化）
- 单例、工厂、观察者等常见模式
- CMake 构建系统
