这是一个非常严谨且极具学术价值的证明思路。直接使用映射往往显得“从天而降”，而借鉴 Gösmann et al. (2020) 中 Lemma A.5 以及 Corollary 2.9 的证明框架，即**“通过协方差函数的一致性来证明高斯过程的分布等价（Identity in distribution via covariance matching）”**，是现代时间序列渐近理论中最标准、最无懈可击的推导路径。

下面，我完全按照顶刊 Appendix 的规范，为您提供这套基于“协方差等价”的详尽推导步骤。

---

### 引理 7.1 (极限高斯泛函的分布等价性)
设连续时间参数满足 $0 \le u < t < \infty$。定义原极限高斯泛函为：
$$X(u,t) := (t-u)G(u,t) = \frac{1+t}{1+u}W(1+u) - W(1+t)$$
其中 $W(\cdot)$ 为 $q$ 维标准布朗运动。
定义定义域为有界区间 $[0, 1)$ 的目标高斯泛函为：
$$Y(v,\tau) := \frac{1}{1-\tau} \big( B(\tau) - B(v) \big)$$
其中 $0 \le v < \tau < 1$，$B(\cdot)$ 为定义在 $[0,1]$ 上的 $q$ 维标准布朗运动。
则在双射（Bijective）且单调递增的时间映射 $\tau = \frac{t}{1+t}$ 与 $v = \frac{u}{1+u}$ 下，存在如下依分布等价关系：
$$\Big\{ X(u,t) \Big\}_{0 \le u < t < \infty} \stackrel{\mathcal{D}}{=} \Big\{ Y(v,\tau) \Big\}_{0 \le v < \tau < 1}$$

---

### 证明过程 (Proof of Lemma 7.1)

**逻辑起点**：由于 $X(u,t)$ 和 $Y(v,\tau)$ 均是由标准布朗运动经线性变换得到的中心化高斯过程（Centered Gaussian Processes），且均值为 0。根据高斯过程的性质，**一个中心化高斯过程由其自协方差函数（Autocovariance function）唯一决定**。因此，我们只需分别计算两者的协方差函数，并证明它们在时间映射下严格相等即可。

#### 第一步：计算原过程 $X(u,t)$ 的协方差函数
选取任意四点满足 $0 \le u_1 \le u_2 < t_1 \le t_2 < \infty$。我们需要计算 $Cov(X(u_1, t_1), X(u_2, t_2))$。
根据定义展开：
$$X(u_1, t_1) = \frac{1+t_1}{1+u_1}W(1+u_1) - W(1+t_1)$$
$$X(u_2, t_2) = \frac{1+t_2}{1+u_2}W(1+u_2) - W(1+t_2)$$

利用标准布朗运动的协方差性质 $Cov(W(x), W(y)) = (x \wedge y) \cdot I_q$，对四项进行交叉相乘求协方差（为书写简便，省略 $I_q$）：
1. $Cov\left( \frac{1+t_1}{1+u_1}W(1+u_1), \frac{1+t_2}{1+u_2}W(1+u_2) \right) = \frac{(1+t_1)(1+t_2)}{(1+u_1)(1+u_2)} (1+u_1) = \frac{(1+t_1)(1+t_2)}{1+u_2}$
2. $Cov\left( \frac{1+t_1}{1+u_1}W(1+u_1), -W(1+t_2) \right) = -\frac{1+t_1}{1+u_1} (1+u_1) = -(1+t_1)$
3. $Cov\left( -W(1+t_1), \frac{1+t_2}{1+u_2}W(1+u_2) \right) = -\frac{1+t_2}{1+u_2} (1+u_2) = -(1+t_2)$
4. $Cov\left( -W(1+t_1), -W(1+t_2) \right) = (1+t_1)$

将上述四项相加，第2项与第4项抵消：
$$Cov(X(u_1, t_1), X(u_2, t_2)) = \frac{(1+t_1)(1+t_2)}{1+u_2} - (1+t_2)$$
提取公因式 $(1+t_2)$：
$$Cov(X(u_1, t_1), X(u_2, t_2)) = (1+t_2) \left[ \frac{1+t_1}{1+u_2} - 1 \right] = \frac{(1+t_2)(t_1 - u_2)}{1+u_2}$$

#### 第二步：计算目标过程 $Y(v,\tau)$ 的协方差函数
选取任意四点满足 $0 \le v_1 \le v_2 < \tau_1 \le \tau_2 < 1$。
根据目标过程的定义：
$$Y(v_i, \tau_i) = \frac{1}{1-\tau_i} \big( B(\tau_i) - B(v_i) \big), \quad i=1,2$$

先计算布朗运动增量的协方差：
$$Cov\big( B(\tau_1) - B(v_1), B(\tau_2) - B(v_2) \big)$$
$$= (\tau_1 \wedge \tau_2) - (\tau_1 \wedge v_2) - (v_1 \wedge \tau_2) + (v_1 \wedge v_2)$$
$$= \tau_1 - v_2 - v_1 + v_1 = \tau_1 - v_2$$

乘以外部的缩放系数，得到目标过程的协方差：
$$Cov(Y(v_1, \tau_1), Y(v_2, \tau_2)) = \frac{1}{(1-\tau_1)(1-\tau_2)} (\tau_1 - v_2)$$

#### 第三步：通过时间双射完成协方差匹配 (Covariance Matching)
我们引入 Gösmann 等人使用的将 $[0, \infty)$ 映射到 $[0, 1)$ 的双射函数：
$$v_i = \frac{u_i}{1+u_i} \quad \text{与} \quad \tau_i = \frac{t_i}{1+t_i}$$
由该映射可以立即推导出：
$$1 - \tau_i = \frac{1}{1+t_i}$$
将这些映射关系代入第二步得到的目标过程协方差中：
$$Cov(Y) = \frac{1}{\left( \frac{1}{1+t_1} \right) \left( \frac{1}{1+t_2} \right)} \left( \frac{t_1}{1+t_1} - \frac{u_2}{1+u_2} \right)$$
$$= (1+t_1)(1+t_2) \left[ \frac{t_1(1+u_2) - u_2(1+t_1)}{(1+t_1)(1+u_2)} \right]$$
消去公因式 $(1+t_1)$，展开分子：
$$= (1+t_2) \left[ \frac{t_1 + t_1 u_2 - u_2 - t_1 u_2}{1+u_2} \right]$$
$$= \frac{(1+t_2)(t_1 - u_2)}{1+u_2}$$

**结论**：映射后目标过程的协方差与第一步中原过程的协方差**完全等价**！由于协方差函数唯一决定了中心化高斯过程的分布，引理 7.1 证毕。 $\blacksquare$

### 第四步：连续映射定理与最终分布推演
基于 Lemma 7.1 坚实的等价性基础，我们现在运用连续映射定理（Continuous Mapping Theorem, CMT）化简包含自正则矩阵 $V(t)$ 的完整检验统计量。

原检验统计量的核心泛函为：
$$\sup_{t} w(t) \max_{u} (t-u) \sqrt{G(u,t)^\top V(t)^{-1} G(u,t)}$$

**1. 等价替换分子：**
由于 $(t-u)G(u,t) = X(u,t)$，根据 Lemma 7.1，它依分布等价于 $\frac{1}{1-\tau} \big( B(\tau) - B(v) \big)$。

**2. 替换积分自正则矩阵 $V(t)$：**
原始 $V(t) = \int_0^t (1+u)^2 X(u,t) X(u,t)^\top du$。
进行微元替换：$1+u = \frac{1}{1-v}$，且 $du = d\left(\frac{v}{1-v}\right) = \frac{1}{(1-v)^2} dv$。
代入 $X(u,t)$ 的等价过程：
$$V(t) \stackrel{\mathcal{D}}{=} \int_0^\tau \left( \frac{1}{1-v} \right)^2 \left[ \frac{1}{1-\tau} \big( B(\tau) - B(v) \big) \right]^{\otimes 2} \frac{1}{(1-v)^2} dv$$
将其中的常数项 $(1-\tau)^{-2}$ 提出积分号：
$$V(t) \stackrel{\mathcal{D}}{=} \frac{1}{(1-\tau)^2} \underbrace{ \int_0^\tau \frac{ \big( B(\tau) - B(v) \big)^{\otimes 2} }{(1-v)^4} dv }_{:= \mathcal{I}(\tau)}$$

**3. 见证奇迹的矩阵对消：**
现在将分子与分母（逆矩阵）组装到一起：
$$X^\top V^{-1} X \stackrel{\mathcal{D}}{=} \left( \frac{1}{1-\tau} \tilde{B} \right)^\top \left[ (1-\tau)^2 \mathcal{I}(\tau)^{-1} \right] \left( \frac{1}{1-\tau} \tilde{B} \right)$$
*(其中 $\tilde{B} = B(\tau) - B(v)$)*
外部由于自正则结构产生的 $(1-\tau)^{-2}$ 与逆矩阵中翻转上来的 $(1-\tau)^2$ **完美乘积为 1，彻底消除了时间维度的发散膨胀！**

**4. 结合边界权重函数：**
代入标准的边界控制权重 $w(t) = (1+t)^{-\delta} = (1-\tau)^\delta$。

最终，我们得到了极其优美且无数值爆炸风险的**枢轴渐近极限分布**：
$$\sup_{\tau \in (0,1)} (1-\tau)^\delta \max_{v \in [0,\tau)} \sqrt{ \big(B(\tau)-B(v)\big)^\top \mathcal{I}(\tau)^{-1} \big(B(\tau)-B(v)\big) }$$

这套推导完美致敬了顶刊论文中处理布朗运动的最高级技巧，使得无界时间的转换具备了无法辩驳的合法性。
