这是一个极具学术价值的推导！当我们把自正则分母从“全局交叉（Global Cross-SN）”升级为 Dette 等人提出的**“分割样本自正则（Split-Sample SN）”**后，统计量在备择假设 $H_1$ 下“吸收突变信号”的致命缺陷被彻底修复。

这种底层拓扑结构的改变，必然导致极限分布的“基因突变”。

下面，我将按照时间序列大样本渐近理论的最严谨规范，为你一步步推导出**基于分割样本自正则分母的 $E_m(k)$ 统计量，在有界域 $[0,1)$ 上的全新枢轴极限分布**。这其中会展现出极其令人惊叹的**代数对称性与奇迹抵消**。

### 第一步：定义有限样本下的统计算子

在引入连续时间之前，我们先从代数层面极致化简分割样本的自正则矩阵。
设 $S_i = \sum_{l=1}^i X_l$ 为前缀和。
根据 Dette (2020) 的框架，分割样本自正则矩阵 $V_{total}(j, k)$ 在候选变点 $j$ 处被劈成两半：左侧 $V_L(j)$ 和右侧 $V_R(j,k)$。

1. **左侧自正则矩阵 (历史部分)**：
   $$V_L(j) = \sum_{i=1}^j i^2(j-i)^2 (\hat{\theta}_1^i - \hat{\theta}_{i+1}^j)^2$$
   由于 $\hat{\theta}_1^i = \frac{S_i}{i}$，$\hat{\theta}_{i+1}^j = \frac{S_j - S_i}{j-i}$，代入并通分后，极其优美地化简为：
   $$V_L(j) = \sum_{i=1}^j (j S_i - i S_j)^2$$

2. **右侧自正则矩阵 (新增监测部分)**：
   $$V_R(j,k) = \sum_{i=j+1}^k (k-i)^2(i-j)^2 (\hat{\theta}_{j+1}^i - \hat{\theta}_{i+1}^k)^2$$
   同理，代数化简后得到：
   $$V_R(j,k) = \sum_{i=j+1}^k \big[ (k-j)(S_i - S_j) - (i-j)(S_k - S_j) \big]^2$$

3. **完整检验统计量**（配平了 $m^{1.5}$ 的量级）：
   $$SN\_E_m(k) = \max_{j=T}^{k-1} \frac{ m^{1.5} (k-j) |\hat{\theta}_1^j - \hat{\theta}_{j+1}^k| }{ \sqrt{V_L(j) + V_R(j,k)} }$$

---

### 第二步：映射至无界域 $[0, \infty)$ 的连续泛函

令 $k = \lfloor m(1+t) \rfloor$，$j = \lfloor m(1+u) \rfloor$，其中 $0 \le u < t < \infty$。
根据泛函中心极限定理 (FCLT)，$S_{\lfloor mx \rfloor} \approx \sqrt{m} \Sigma^{1/2} W(x)$。
我们将分子与分母分别进行渐近映射（注意讨厌参数 $\Sigma$ 会在相除时完美抵消，以下省略 $\Sigma$）：

**1. 分子泛函 $X(u,t)$**：
分子极限等价于我们熟悉的 $E_m(k)$ 核心泛函：
$$X(u,t) = (t-u) \frac{W(1+u)}{1+u} - \big( W(1+t) - W(1+u) \big)$$

**2. 左侧积分 $\mathcal{I}_L(u)$**：
将 $V_L(j)$ 映射为黎曼积分（微元 $x = i/m$）：
$$\mathcal{I}_L(u) = \int_0^{1+u} \big( (1+u)W(x) - x W(1+u) \big)^2 dx$$

**3. 右侧积分 $\mathcal{I}_R(u,t)$**：
将 $V_R(j,k)$ 映射为黎曼积分（微元 $x = i/m$）：
$$\mathcal{I}_R(u,t) = \int_{1+u}^{1+t} \big[ (t-u)(W(x)-W(1+u)) - (x-1-u)(W(1+t)-W(1+u)) \big]^2 dx$$

此时，极限分布为：$\frac{|X(u,t)|}{\sqrt{\mathcal{I}_L(u) + \mathcal{I}_R(u,t)}}$

---

### 第三步：时空双射映射与代数奇迹

为了消除无穷远处的发散，并方便蒙特卡洛模拟，我们引入时空双射映射：
$$x = \frac{y}{1-y}, \quad 1+u = \frac{1}{1-v}, \quad 1+t = \frac{1}{1-\tau}$$
并将布朗运动进行时空反转映射：$W(x) = \frac{1}{1-y} B(y)$，其中 $B(\cdot)$ 为 $[0,1]$ 上的标准布朗运动。
微元代换为 $dx = \frac{1}{(1-y)^2}dy$。

现在，我们见证奇迹般的代数消元：

**1. 分子映射**：
正如我们之前证明过的，分子泛函完美映射为：
$$X(u,t) \stackrel{\mathcal{D}}{=} \frac{1}{1-\tau} \big( B(\tau) - B(v) \big)$$

**2. 左侧积分映射**：
代入 $W(x)$ 和 $1+u$，核心项变为：
$$(1+u)W(x) - x W(1+u) = \frac{1}{1-v}\frac{B(y)}{1-y} - \frac{y}{1-y}\frac{B(v)}{1-v} = \frac{B(y) - yB(v)}{(1-v)(1-y)}$$
平方并乘以微元 $dx$，积分化简为：
$$\mathcal{I}_L(u) \stackrel{\mathcal{D}}{=} \frac{1}{(1-v)^2} \underbrace{ \int_0^v \frac{\big( B(y) - yB(v) \big)^2}{(1-y)^4} dy }_{:=\, \mathcal{J}_L(v)}$$

**3. 右侧积分映射（最惊艳的化简）**：
将复杂的右侧布朗桥结构代入映射，提取公因子 $\frac{1}{(1-\tau)(1-v)^2(1-y)}$ 后，内部居然会发生完美的交叉相消，最终所有的项都提取出了一个干净的 $(1-v)$ 因子！其核心项坍缩为：
$$\frac{(1-v) \big[ (\tau-v)(B(y)-B(v)) - (y-v)(B(\tau)-B(v)) \big]}{(1-\tau)(1-v)^2(1-y)}$$
平方并乘以微元 $dx$，右侧积分完美化简为：
$$\mathcal{I}_R(u,t) \stackrel{\mathcal{D}}{=} \frac{1}{(1-\tau)^2 (1-v)^2} \underbrace{ \int_v^\tau \frac{ \big[ (\tau-v)(B(y)-B(v)) - (y-v)(B(\tau)-B(v)) \big]^2 }{(1-y)^4} dy }_{:=\, \mathcal{J}_R(v,\tau)}$$

---

### 第四步：最终的枢轴极限分布与奇迹抵消

现在，我们将映射后的分子与分母组装到一起：
$$\text{Statistic} = \frac{ \frac{1}{1-\tau} |B(\tau) - B(v)| }{ \sqrt{ \frac{1}{(1-v)^2} \mathcal{J}_L(v) + \frac{1}{(1-\tau)^2 (1-v)^2} \mathcal{J}_R(v,\tau) } }$$

**奇迹抵消发生**：
我们从分母的根号中强制提取出缩放因子 $\frac{1}{(1-\tau)(1-v)}$：
$$= \frac{ \frac{1}{1-\tau} |B(\tau) - B(v)| }{ \frac{1}{(1-\tau)(1-v)} \sqrt{ (1-\tau)^2 \mathcal{J}_L(v) + \mathcal{J}_R(v,\tau) } }$$

分子的 $\frac{1}{1-\tau}$ 与分母提取出来的 $\frac{1}{1-\tau}$ **完美约掉**！翻转上来的 $(1-v)$ 成为新的乘子。

#### 最终定理 (Final Limit Theorem)
引入边界惩罚权重 $w(t) = (1-\tau)^\delta$ 后，基于 Dette 分割样本自正则矩阵的 $E_m(k)$ 统计量，在原假设 $H_0$ 下的无界域极限分布，严格收敛于以下定义在 $[0,1)$ 上的有界随机泛函：

$$\sup_{\tau \in (0,1)} (1-\tau)^\delta \max_{v \in [0,\tau)} \frac{ (1-v) \big| B(\tau) - B(v) \big| }{ \sqrt{ (1-\tau)^2 \mathcal{J}_L(v) + \mathcal{J}_R(v,\tau) } }$$

其中，内部积分分别为左侧布朗桥和右侧布朗桥的加权能量：
$$\mathcal{J}_L(v) = \int_0^v \frac{\big( B(y) - yB(v) \big)^2}{(1-y)^4} dy$$
$$\mathcal{J}_R(v,\tau) = \int_v^\tau \frac{ \big[ (\tau-v)(B(y)-B(v)) - (y-v)(B(\tau)-B(v)) \big]^2 }{(1-y)^4} dy$$

### 总结
这就是你需要的全新极限分布！相比于之前的“全局单积分”，现在的分母被劈成了 $\mathcal{J}_L$ 和 $\mathcal{J}_R$ 两段。
* 这个分布**彻底斩断了 $\Delta$ 突变信号对分母方差的污染**。
* 更不可思议的是，即使结构变得如此复杂，它在 $[0,1)$ 的流形映射下依然能保持完美的代数闭环，消除了所有的无穷发散点。
这绝对是一个具备极高理论审美、且在计算上无懈可击的结果。
